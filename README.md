# MySLAVA

MySLAVA — это кроссплатформенное настольное приложение на Python (PyQt 6 + Pyrogram) для массовой рассылки сообщений в Telegram с нескольких аккаунтов.

## Возможности
* Добавление неограниченного числа аккаунтов по API ID / API Hash Telegram.
* Удобный редактор шаблонов сообщений (HTML → Markdown V2).
* Хранение шаблонов в локальной директории `scripts/`.
* Нацеливание рассылки на отдельные списки получателей для каждого аккаунта.
* Случайная задержка между сообщениями (анти-спам).
* Подробный HTML-лог каждой рассылки в `broadcast_logs/`.
* Все данные хранятся **локально** в директории пользователя (см. ниже).

## Быстрый старт
1. Установите Python 3.10 или новее.
2. Клонируйте репозиторий и создайте виртуальное окружение:

```bash
python3 -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

4. Запустите приложение:

```bash
python main.py
```

## Сборка standalone-приложения
Используется PyInstaller. В репозитории есть готовые спек-файлы:

```bash
pyinstaller myslava.spec         # macOS
pyinstaller slava_aig_win.spec   # Windows
```

Собранный `.app` / `.exe` будет лежать в директории `dist/`.

## Где хранятся пользовательские данные
Все рабочие файлы помещаются в каталог пользователя:

* **macOS:** `~/Library/Application Support/SLAVA`
* **Windows:** `%APPDATA%\SLAVA`

Внутри него создаются:

* `accounts.json` — сохранённые аккаунты Telegram
* `broadcast_logs/` — HTML-отчёты о рассылках
* `scripts/` — шаблоны сообщений
* `sessions/` — файлы сессий Pyrogram

Эти директории добавлены в `.gitignore` и не попадают в репозиторий.
